### 项目结构
src/                项目源码
    assets/         资源（图片，字体，icon等）
    components/     公共组件
    pages/          页面
    utils/          工具
    APP.js          根组件（配置路由信息）
    index.css       全局样式
    index.js        项目入口文件

### 项目说明
1. 基于React, antd 框架
2. 数据源： data.json, mockjs
3. api方面未使用router

### 启动说明   
1. prepare:
   - npm install
   - cd src
   - npx json-server singers.json 
2.  npm start
